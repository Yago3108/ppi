CSS é um documento de estilização em cascata criado em 1993,com sua primeira versão amplamente utilizada em 1997. A linguagem CSS permite definir regras que define características para os elementos HTML como cor,borda e comportamentos, além de animações e transições.


Ao longo dos anos o css evoluiu e hoje temos o css 1 2 e 3, cada uma das versões adicionou novos recursos que eram demandados no período

  1. CSS 1 1997 - propriedades de estilização e formatação básicas,além de controle do posicionamento dos elementos html com display,propriedades float e position
  2. CSS 2 2000 - adição de propriedades de animação e transição e melhoria do position
  3. CSS 3 2008 - incremento gradual de novas funcionalidades do CSS com animação e ferramentas de posicionamento como display:flex (2008) e grid (2013)
